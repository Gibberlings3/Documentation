Improved Horns of Valhalla

A Gibberlings Three Mod
Author: Eric Kerr (aka Cirerrek)

Version 2 - Check for the most recent version
Languages: English, French, German, Italian, Polish
Platforms: Windows, Mac OS X, and Linux

Overview

This is a minor modification to Baldur's Gate II that improves the berserkers summoned by the three Horns of Valhalla. Only the creatures and their items are modified. The horns themselves are left unchanged. The berserkers have improved weaponry, proficiencies, and are constantly in a state of enrage (per the Berserker kit).

Compatibility

This mod is designed to work with BG2: Shadows of Amn with or without the expansion pack Throne of Bhaal as well as the Enhanced Editions of the game. At present there are no known incompatibilities.

Installation

Windows
Improved Horns of Valhalla is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a valhorn folder and setup-valhorn.exe in your game folder. To install, simply double-click setup-valhorn.exe and follow the instructions on screen.

Please run setup-valhorn.exe in your game folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Improved Horns of Valhalla is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your game folder. If properly extracted, you should have a valhorn folder, setup-valhorn, and setup-valhorn.command in your game folder. To install, simply double-click setup-valhorn.command and follow the instructions on screen.

Linux
Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables in a $PATH directory (E.G. /usr/bin). Following that, open a terminal, cd to your game installation directory, run `tolower' and answer Y to both queries.

You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you're unsure, running tolower and choosing both options is the safe bet.

After that, run 'weinstall valhorn' from your game installation directory to install or uninstall the mod.

Contents

All three of the berserkers summoned by the horn are berserk as per the Berserker Fighter Kit Innate ability BERSERKER_ENRAGE for the entire duration that they are in existence (60 seconds).

This is done through the replacement of the item BERSERK.ITM with the item BERSERKR.ITM

BERSERKR.ITM provides the following advantages/disadvantages:

  • Berserk: Not exactly sure what this does but the original item being replaced had it and MINSC_BERSERK also has it
  • Attack Damage Bonus: +2
  • AC: +2 (Note: This is a disadvantage)
  • Immunity to Stun
  • Immunity to Charm Creature
  • Immunity to Panic
  • Immunity to Maze
  • Immunity to Hold Creature Type
  • Immunity to Confusion
  • Immunity to Sleep
  • Immunity to Hold Creature
  • Immunity to Feeblemindedness
  • Immunity to Break Morale
  • Immunity to Level Drain
  • Immunity to Imprisonment
  • Max HP Bonus: +15
  • Modify Scripting State - SCRIPTINGSTATE4 Value 1 (Allows berserk state to be detectable through scripting)
  • THAC0 Bonus: +2
