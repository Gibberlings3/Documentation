Totemic Cernd

A Gibberlings Three Mod
Author: Eric Kerr (aka Cirerrek)

Version 3 - Check for the most recent version
Languages: English, French, German, Italian, Polish
Platforms: Windows, Mac OS X, and Linux

Overview

Totemic Cernd is a minor modification to Baldur's Gate II (original or Enhanced Editions) that changes Cernd's druid kit from that of a Shapeshifter to a Totemic Druid. In the process, I also decided to increase Cernd's physical statistics somewhat to make him a reasonably attractive alternative to other clerics or druids, such as Jaheira or Anomen.

Compatibility

This mod is designed to work with BG2: Shadows of Amn with or without the expansion pack Throne of Bhaal and with the Enhanced Edition of same. At present there are no known incompatibilities.

Installation

Windows
Totemic Cernd is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a totemiccernd folder and setup-totemiccernd.exe in your game folder. To install, simply double-click setup-totemiccernd.exe and follow the instructions on screen.

Please run setup-totemiccernd.exe in your game folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Totemic Cernd is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your game folder. If properly extracted, you should have a totemiccernd folder, setup-totemiccernd, and setup-totemiccernd.command in your game folder. To install, simply double-click setup-totemiccernd.command and follow the instructions on screen.

Linux
Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables in a $PATH directory (E.G. /usr/bin). Following that, open a terminal, cd to your game installation directory, run `tolower' and answer Y to both queries.

You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you're unsure, running tolower and choosing both options is the safe bet.

After that, run 'weinstall totemiccernd' from your game installation directory to install or uninstall the mod.

Contents

Totemic Cernd is a minor modification to Baldur's Gate II that changes Cernd's druid kit from that of a Shapeshifter to a Totemic Druid. In the process, I also decided to increase Cernd's physical statistics somewhat to make him a reasonably attractive alternative to other clerics or druids, such as Jaheira or Anomen.

Fortunately for me, not one dialog in the entire game mentions the fact that Cernd is a Shapeshifter. So, no in-game dialogues had to be altered, which made this modification a relatively easy one in terms of the amount of files I had to edit.

Altered Files:

.CRE

  • CERND10.CRE
  • CERND12.CRE
  • CERND13B.CRE
  • CERND14.CRE

.BCS

  • AR1901.BCS
  • CEFALD01.BCS
  • CUT35I.BCS

.ITM

  • NPSTAF.ITM - Staff of the High Forest
  • NPCLCK.ITM - Cloak of the High Forest

The adjustments to the scripts are primarily to deal with the situation when Cernd is not in the party and challenges Faldorn to ritual combat for the Druid Grove. Faldorn's script causes Cernd to transform into a special werewolf with the MINHIP1.ITM, meaning it is impossible for Cernd to lose the fight. The script changes prevent the transformation from occurring and also gives him the MINHP1.ITM temporarily.

Attribute Previous New
Strength 13 16
Dexterity 9 17
Constitution 13 16
Intelligence 12 12
Wisdom 18 18
Charisma 15 15
Total 80 94

An increase of 14 attribute points is a large jump, but I believe it can be justified by looking at the attribute totals of the Shapeshifter Werewolf form, WEREWODR.CRE (89) and the Shapeshifter Greater Werewolf form, WEREGR01.CRE (105).
