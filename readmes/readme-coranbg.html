Notes

This mod is designed for use on the following platforms: Tutu, BGT and Baldur's Gate Enhanced Edition (BGEE).

It features an installer using cross-platform variables to install the same code on Tutu, BGT and BGEE installations. For more information on how you can use CrossPlatform variables in your mod, please visit us on our forums.

Overview

This mod expands Coran's Friendship talk sequence, adding up to eighteen (18) new banters with the protagonist (and one potential comment by Garrick) for a player character who is not in an active romance with him. If you have The BG1 NPC Project installed, they begin playing after the last regular friendship talk. If the player started and interrupted Coran's The BG1 NPC Project romance, the friendship sequence should initiate after that. If you do not have The BG1 NPC Project installed, they become available immediately.

Compatibility

This mod is designed to work on Tutu v4, EasyTutu_SoA, EasyTutu_ToB, and BGT, all based on full, patched installs of Baldur's Gate and Baldur's Gate II. Neither the BG2 expansion Throne of Bhaal nor the BG expansion Tales of the Sword Coast are required. As of v4, the mod is also compatible with BGEE.

This mod will not install correctly under versions of WeiDU below v237. The setup-coranbgfriend.exe distributed with this mod will automatically update your WeiDU installer version when you install this mod.

It is unlikely to be compatible with Tutu v6. It might install, but may behave incorrectly. Don't say we didn't warn you.

Installation

Windows

Coran's BG Extended Friendship Talks for Tutu and BGT is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a "coranbgfriend" folder, and setup-coranbgfriend.exe in your game folder. To install, double-click setup-coranbgfriend.exe and follow the instructions on screen.

Please run setup-coranbgfriend.exe in your game folder to reinstall, uninstall or otherwise change components.

Mac OS X

If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. This mod is packaged and installed with WeiDU. If properly extracted, you should have a "coranbgfriend" folder, "setup-coranbgfriend", and "setup-coranbgfriend.command" in your game folder. To install, simply double-click setup-coranbgfriend.command and follow the instructions on screen.

Linux

Extract the contents of the mod to the folder of the game you wish to modify. If not packaged with the mod, download the latest version of WeiDU and copy WeiDU and WeInstall to /usr/bin. Run WeInstall coranbgfriend in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Install Order

While this mod is very independent of install order issues, adding banters and not touching other existing dialogue states, it does search for The BG1 NPC Project. If you are planning to install The BG1 NPC Project, please install it before Coran's BG Extended Friendship Talks for Tutu, BGT and BGEE.

Modder's Notes

All of the content in this mod is added nondestructively to Coran's and Garrick's Baldur's Gate content. All content is shut down in a BGT using the blocking variable ~!Global("endofbg1","GLOBAL",2)~. This variable is set via OUTER_SPRINT and will not appear in Tutu games.
