Zakrion NPC Mod for Baldur's Gate: Enhanced Edition

A Gibberlings Three Mod
Author: jmaeq
On the web: Home page and discussion forum

Version 1.0
Languages: English
Platforms: Windows, Mac OS X, Linux

Overview

Zakrion introduces a human Duelist to BG:EE. Zakrion can be found near the ship wreckage on the Sword Coast. The mod includes custom portraits and voicing, a friendship track, numerous banters and interjections, and a "semi-romance" with the PC.

Compatibility

This mod is designed to work with Baldur's Gate: Enchanced Edition by Bioware and enhanced by Beamdog.

The mod should be compatible with other WeiDU mods.

Bug Reporting

If you should encounter any bugs, please report them to the authors at the Zakrion forum.

Installation

Windows
Zakrion_BG1 is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a Zakrion_BG1 folder and Setup-Zakrion_BG1.exe in your Tutu folder. To install, simply double-click Setup-Zakrion_BG1.exe and follow the instructions on screen.

Please run Setup-Zakrion_BG1.exe in your Tutu folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Zakrion_BG1 is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your Tutu folder. If properly extracted, you should have a Zakrion_BG1 folder, Setup-Zakrion_BG1, and Setup-Zakrion_BG1.command in your Tutu folder. To install, simply double-click Setup-Zakrion_BG1.command and follow the instructions on screen.

Linux
Extract the contents of the mod to the folder of the game you wish to modify. Download the latest version of WeiDU for Linux from WeiDU.org and copy WeiDU and WeInstall to /usr/bin. Following that, open a terminal, cd to your BG2 installation directory, run 'tolower' and answer Y to both queries. You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you're unsure, running tolower and choosing both options is the safe bet.
Run WeInstall zakrion_bg1 in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Contents

This modification adds Zakrion, a True Neutral Human Duelist, to the game. His statistics are as follows:

  • STR: 16
  • DEX: 17
  • CON: 15
  • INT: 10
  • WIS: 12
  • CHR: 14

He has a proficiency in two weapon style, short swords, and long swords.

Zakrion includes a friendship track, a semi-romance with the PC, banters with all Bioware NPCs (including Beamdog's four NPCs), and numerous interjections.
