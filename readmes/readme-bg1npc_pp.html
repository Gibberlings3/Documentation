Overview

This mod provides alternate portraits for the Baldurs Gate NPCs.

Compatibility

This mod is designed to work with Baldur's Gate, with or without the expansion Tales of the Sword Coast. It also supports several conversion projects, BGT(Baldur's Gate II: Shadows of Amn) with or without the Throne of Bhaal expansion pack, Tutu v4, EasyTutu_SoA, and EasyTutu_ToB. The mod should be compatible with other WeiDU mods.

Installation

Windows
The BG1 NPC Portrait Pack is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a "bg1npc_portrait_pack" folder and setup-bg1npc_portrait_pack.exe in your game folder. To install, simply double-click setup-bg1npc_portrait_pack.exe and follow the instructions on screen.

Please run setup-bg1npc_portrait_pack.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. "bg1npc_portrait_pack" is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your game folder. If properly extracted, you should have a "bg1npc_portrait_pack" folder, setup-bg1npc_portrait_pack, and setup-bg1npc_portrait_pack.command in your game folder. To install, double-click setup-bg1npc_portrait_pack.command and follow the instructions on screen.

Linux
Extract the contents of the mod to the folder of the game you wish to modify. Download the latest version of WeiDU for Linux from WeiDU.org and copy WeiDU and WeInstall to /usr/bin. Following that, open a terminal, cd to your BG2 installation directory, run 'tolower' and answer Y to both queries. You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you're unsure, running tolower and choosing both options is the safe bet.
Run WeInstall bg1npc_portrait_pack in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Contents

The mod contains separate components that can be installed independently of one another.

Copy all portraits for use by player characters
This component will copy all portraits included with the mod into your portraits folder for use by your PC.

Replace all NPC portraits (AMOK)
This component will replace all NPC portraits with this set by AMOK .

Alternate Branwen portrait: BioWare edit by XVIII
An editied version of Branwen's original portrait by XVIII.

Alternate Coran Portraits
You have your choice of three Coran alternates, one by Melisah, a second from NiGHTMARE and Tancred and a third by Amok.

Alternate Dynaheir Portraits
You have your choice of three Dynaheir alternates by Amok.

Alternate Faldorn Portrait: by XVIII
An new Faldorn portrait by XVIII.

Alternate Kivan Portrait: BioWare edit by XVIII
An editied version of Kivan's original portrait by XVIII.

Alternate Safana Portrait: BioWare edit by XVIII
An editied version of Safana's original portrait by XVIII.

Alternate Viconia Portrait: BioWare edit by XVIII
An editied version of Viconia's original portrait by XVIII.
