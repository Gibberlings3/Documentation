Overview

The mod adds an elven fighter to BG2, BGT, or BGII:EE. Haldamir can be found inside the Den of the Seven Vales in Waukeen's Promenade.

Compatibility

This mod is designed to work with Baldur's Gate II: Shadows of Amn with or without the Throne of Bhaal expansion pack, BGT, or BGII:EE. The mod should be compatible with other WeiDU mods.

Installation

Windows
Haldamir is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a haldamir folder and setup-haldamir.exe in your BG2 folder. To install, simply double-click setup-haldamir.exe and follow the instructions on screen.

Please run setup-haldamir.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Haldamir is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 directory.  If properly extracted, you should have a haldamir folder, setup-haldamir, and setup-haldamir.command in your BG2 directory. To install, simply double-click setup-haldamir.command and follow the instructions on screen.

Linux
Extract the contents of the mod to the folder of the game you wish to modify. Download the latest version of WeiDU for Linux from WeiDU.org and copy WeiDU and WeInstall to /usr/bin. Following that, open a terminal, cd to your BG2 installation directory, run 'tolower' and answer Y to both queries.
You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you are unsure, running tolower and choosing both options is the safe bet.
Run WeInstall haldamir in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Contents

This mod adds Haldamir, an unkitted elven fighter to Baldur's Gate II: Shadows of Amn. Haldamir can be found at the Den of the Seven Vales in Waukeen's Promenade. The mod offers two banters with each BioWare NPC, five banters with a non-elven PC (six with an elven PC), scenery dialogue, and over a hundred interjections. There is no quest and no romance, but there is a friendship, of sorts.
