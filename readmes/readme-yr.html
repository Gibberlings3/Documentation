Introduction

Was that truly the ending to Yoshimo's adventures with the Bhaalspawn? Is there simply no way to redeem himself or explain his actions to the PC?

Yoshimo's Remorse is a mod for BG2EE that makes Yoshimo a playable character in Throne of Bhaal. However, he is not what you expect him to be...

Content

Yoshimo's Remorse includes:

  • 7 timered "friendship" talks
  • 3 additional talks upon mentioning former romantic feelings
  • Talks fired by circumstances
  • Multiple epilogues depending on your actions
  • 2 banters with all BioWare NPCs and Beamdog NPCs
  • Interjections
  • New Items
  • Cutscenes
  • An encounter with a new minor character

Installation

First time installing a mod? Check out G3's comprehensive tutorial: A New Player's Guide to Installing and Playing Mods.

Enhanced Editions Note
The Enhanced Editions are actively supported games. Please note that every patch update will wipe your current mod setup! If in the middle of a modded game you might want to delay the patch update (if possible) as even after reinstalling the mods, you might not be able to continue with your old savegames. Alternatively, copy the whole game's folder into a new one that can be modded and will stay untouched by game patches. It is important that you install the mod to the language version you are playing the game in. Otherwise, the dialogues of the mod will not show but give error messages.

Windows
Yoshimo's Remorse for Windows is distributed as a zip file and the files can be extracted into your game directory using 7zip or WinRAR. When properly extracted, your game directory will contain setup-yoshimosremorse.exe and the folder yoshimosremorse. To install, double-click setup-yoshimosremorse.exe and follow the instructions on screen.

You can run setup-yoshimosremorse.exe in your game folder to reinstall, uninstall or otherwise change components.

macOS
Yoshimo's Remorse for macOS is distributed as a compressed tarball and includes a WeiDU installer.

First, extract the files from the tarball into your game directory. When properly extracted, your game directory will contain setup-yoshimosremorse, setup-yoshimosremorse.command, and the folder yoshimosremorse. To install, double-click setup-yoshimosremorse.command and follow the instructions on screen.

You can run setup-yoshimosremorse.command in your game folder to reinstall, uninstall or otherwise change components.

Linux
Yoshimo's Remorse for Linux is distributed as a compressed tarball and does not include a WeiDU installer. Linux users will need to do a one-time install of WeiDU (and a few other adjustments) as described in this great writeup.

To install, run 'WeInstall yoshimosremorse in your game folder.

Note for Complete Uninstallation
In addition to the methods above for removing individual components, you can completely uninstall the mod using setup-yoshimosremorse --uninstall at the command line to remove all components without wading through prompts.

Compatibility

This mod is designed to work with Baldur's Gate II: Enhanced Edition.

The mod should be compatible with other WeiDU mods. However, it is incompatible with Keeping Yoshimo.
