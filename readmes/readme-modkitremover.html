Mod Kit Remover

Version 3 - Check for the most recent version

The Mod Kit Remover is a WeiDU mod that will remove all mod kits from the character creation screen. The mod was primarily made because The Darkest Day fills all available kit slots, but players wished to use kits from other mods when playing it. There are options to remove either just mod kits or BioWare and mod kits. All kits will remain installed for use by NPCs and previously created PCs. Any changes made by other mods to BioWare's kits will remain intact, and kits from other WeiDU mods can be installed and be selectable at character creation.

Compatibility

This mod is designed to work with Baldur's Gate II: Shadows of Amn with or without the Throne of Bhaal expansion pack. Aside from the obvious potential incompatibilities with mods that add kits there should be no problems with other mods. Please ensure that you only install this mod when you are sure that you will no longer want to create a character with a previously installed mod kit.

Installation

Windows
The Mod Kit Remover is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a ModKitRemover folder and Setup-ModKitRemover.exe in your BG2 folder. To install, simply double-click Setup-ModKitRemover.exe and follow the instructions on screen.

Please run Setup-ModKitRemover.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. The Mod Kit Remover is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder. If properly extracted, you should have a ModKitRemover folder, Setup-ModKitRemover and Setup-ModKitRemover.command in your BG2 folder. To install, simply double-click Setup-ModKitRemover.command and follow the instructions on screen.

Contents

The Mod Kit Remover contains separate components that can be installed independently of one another. Each component contains the option to remove either just mod kits or BioWare and mod kits.
