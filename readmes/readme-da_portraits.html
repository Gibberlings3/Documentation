Overview

This mod offers alternative portraits for each of the BioWare Baldur's Gate and Baldur's Gate II NPCs. It also offers a selection of alternative portraits for mod NPCs. These portraits can also be used by the PC.

Compatibility

This mod is designed to work with Baldur's Gate or Baldur's Gate II: Shadows of Amn, with or without the Throne of Bhaal expansion. It may also be used with the Baldur's Gate conversion, Tutu, with or without the Tales of the Sword Coast expansion pack, or with BGT-WeiDU. This mod should be installed in your Baldur's Gate, Baldur's Gate Tutu, or Baldur's Gate II directory. For many users, this directory is either Black Isle/Baldur's Gate or Black Isle/BGII - SoA. Tutu users should install onto Tutu after you have run the conversion program, in your Baldur's Gate Tutu directory, or in your BG2 directory for BGT. Please note that for EasyTutu users, this directory is usually named BaldursGateTutu.

There are no known incompatibilities with any mod, however any portrait modification mod will overwrite the portrait assignments of portrait mods installed previously.

This mod should be installed after all mods that add NPCs.

Installation

Windows
berelinde's DA Portraits is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a "da_portraits" folder and setup-da_portraits.exe in your game directory. To install, simply double-click setup-da_portraits.exe and follow the instructions on screen.

Please run setup-da_portraits.exe in your game directory to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. berelinde's DA Portraits is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your game directory. If properly extracted, you should have a "da_portraits" folder, setup-da_portraits, and setup-da_portraits.command in your game directory. To install, simply double-click setup-da_portraits.command and follow the instructions on screen.

Contents

The portraits can be previewed on the Dragon Age: Origins Style Portraits page.

PC Portraits: This component allows the PC to use any of the portraits included in this mod.

BG2 Portraits: This component offers up to four alternate portraits for each of the BG, BG2, or mod NPCs.
