A large number of the Baldur's Gate inventory graphics (BAMs) were improved for the sequel, so I decided to create this small pack that puts them back into the original. It will also work with Icewind Dale, as that re-uses many BG items.

- NiGHTMARE

Compatibility

This should be compatible with all Baldur's Gate and Icewind Dale expansions and mods.

Installation

Windows
Improved Inventory BAMs is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have an ImprovedBAMs folder and Setup-ImprovedBAMs.exe in your BG/IWD folder. To install, simply double-click Setup-ImprovedBAMs.exe and follow the instructions on screen.

Please run Setup-ImprovedBAMs.exe in your BG/IWD folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. Improved Inventory BAMs is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG/IWD folder. If properly extracted, you should have an ImprovedBAMs folder, Setup-ImprovedBAMs, and Setup-ImprovedBAMs.command in your BG/IWD folder. To install, simply double-click Setup-ImprovedBAMs.command and follow the instructions on screen.
