WHAT IS SAPP?

SAPP is sporeboy!'s Alternate Portrait Pack. It provides alter portraits for the female BioWare NPCs in BG2: Aerie, Imoen, Jaheira, Mazzy, Nalia and Viconia. You also get an alternate Valen portrait for Wes Weimer's Valen mod. All of these portraits are optional.

INSTALLING SAPP

Windows
sporeboy!'s Alternate Portrait Pack is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have an sapp folder, Setup-sapp.tp2 and Setup-sapp.exe in your BG2 folder. To install, simply double-click Setup-sapp.exe and follow the instructions on screen.

Please run Setup-sapp.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. sporeboy!'s Alternate Portrait Pack is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder.  If properly extracted, you should have an sapp folder, Setup-sapp.tp2, Setup-sapp, and Setup-sapp.command in your BG2 folder. To install, simply double-click Setup-sapp.command and follow the instructions on screen. You'll receive a message informing you that WeiDU can not auto-update--don't worry, this is normal.

COMPATIBILITY

This mod does not require Throne of Bhaal and there is no reason why this mod should conflict with other mods.
