iSeries

A Gibberlings Three Mod
Author: Eric Kerr (aka Cirerrek)

Version 0.6 Beta- Check for the most recent version

Overview

The iSeries is a set of very beta versions of my scripts for Icewind Dale II. So far the scripts only use a limited number of spells. Otherwise they appear to do most of the other things the eSeries was designed to do: Threat Level Assessment, Weapon Switching, Move to Combat, and a few other things. They're NOT going to be able to strut their stuff with a high level party. Packaging is very basic. Constructive feedback is encouraged. Use iBard for Bards, iArcher for Mages, and iFighter for Fighter types or Cleric types.

Compatibility

This mod is designed to work with Icewind Dale II. There are no known incompatibilities at present.

Installation

Windows
The iSeries is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a iseries folder and Setup-iSeries.exe in your IWD2 folder. To install, simply double-click Setup-iSeries.exe and follow the instructions on screen.

Please run Setup-iSeries.exe in your IWD2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. The iSeries is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your IWD2 folder.  If properly extracted, you should have a iseries folder, Setup-iSeries, and Setup-iSeries.command in your IWD2 folder. To install, simply double-click Setup-iSeries.command and follow the instructions on screen.
