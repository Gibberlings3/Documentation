A Gibberlings Three Mod

Principal Authors: CamDawg, DavidW

Contributing Authors: LadeJarl (GUI), Nythrun (spell and projectile conversion and miscellaneous coding advice)

Languages: English

Overview

This mod converts Icewind Dale to run in the Baldur's Gate II engine, permitting dual wielding, kits, sorcerers, monks, barbarians and the like, and offering a better platform for joinable NPCs.

The mod requires a complete install of Icewind Dale, Heart of Fury, Test of the Luremaster, Baldur's Gate II, and Throne of Bhaal.

The mod is currently in BETA and certainly contains bugs. Please report any you find at the Gibberlings 3 forums.

Compatibility

For obvious reasons, Icewind Dale in Baldur's Gate II is incompatible with most mods for Baldur's Gate II. It is known to be compatible with the Widescreen Mod; there is a reasonable chance that it will be compatible with tweak or kit mods.

Icewind Dale in Baldur's Gate II is incompatible with any Icewind Dale mod that has not been specifically designed for it. At present, the only such mods are the IWD-in-BG2 Fixpack/Tweakpack (available at www.gibberlings3.net; should be installed straight after the main mod), Kulyok's "Icewind Dale NPCs" (available at www.pocketplane.net and CamDawg's "Unfinished Business for Icewind Dale" and "Icewind Dale Item Upgrades" (available at www.gibberlings3.net).

Languages

At present, IWD-in-BG2 itself is in English only. However, it should be compatible with non-English versions of IWD. Other than a very small number (fewer than 10) of strings that are in English, it should work fine in other languages.

Installation

You will need:

  • A full, unmodded install of Icewind Dale, together with the Heart of Winter expansion and the Test of the Luremaster downloadable expansion. (The Good Old Games version has all of this.)
  • A full, unmodded install of Baldur's Gate II, together with the Throne of Bhaal expansion. (Actually, a modded one will probably work too, but I don't guarantee it.)
  • A Windows PC with several GB of spare space. (I don't know exactly how much is needed. 10 is fine.)

Run the Icewind Dale in Baldur's Gate II program (as an administrator for preference, if using Vista or Windows 7) and extract the files to a new directory. (On Vista or Windows 7, avoid choosing a directory in the "Program Files" folder). When prompted, enter the paths to your Baldur's Gate II installation. After a couple of minutes, you will be prompted to enter your Icewind Dale installation; do so. Then go and get a coffee. (Installation will take 15 minutes or more, depending on your computer.)

Important: about 5 minutes into the installation, the final movie of Heart of Winter will play. This is intentional. Since it contains spoilers, turn the sound down and look away in the (unlikely) event that you haven't already seen it.

After the mod has installed, you should download and install the "Icewind-Dale-in-Baldur's-Gate-II Fixpack/Tweakpack", from www.gibberlings3.net. The first component of this contains any current hotfixes to the mod. The rest contains options to customise your installation, mostly concerning how much you want the mod to play like Icewind Dale and how much like Baldur's Gate II.

Your Party

By default, IWD-in-BG2 (because it uses the BG2 engine) assumes that you will generate a single PC. This will make the game fairly difficult. Your options for a party are:

  1. Create a multi-player party, like the original IWD. To do this, start a multi-player game and create your party. As soon as you start the game, save, and then quit the game. Move your savegame folder from the "MPSAVE" directory to the "SAVES" directory. Then restart the game and load this save as a single-player game. This will lead to a rather smoother game experience.
  2. Use the basic NPCs available in the IWD-in-BG2 Fixpack/Tweakpack. This gives you access to 11 basic NPCs, created using IWD's portraits and soundsets. They're slightly more detailed than typical BG1 NPCs, but not nearly as detailed as BG2 NPCs.
  3. Download a mod or mods that add fully developed BG2-style NPCs. At present there is only one such mod: Kulyok's "Icewind Dale NPCs", which adds five NPCs to the game and is compatible with IWD-in-BG2.

Note that by default, the game ends only when the entire party is dead, and NPCs initiate conversation with whichever party member talks to them (i.e., not necessarily with your main character). This is to mirror IWD behaviour; it can, however, be modified using the Fixpack/Tweakpack.

Known Issues

These are the main issues (a more complete list can be found at the mod's forums).

  • A small number of spells cannot currently be converted at all. The current list is: Seven Eyes, Soul Eater, (Great) Shout, Mordenkainen's Force Missiles, Spiritual Wrath, Mold Touch, Wall of Moonlight, and Smashing Wave. Items that cast these spells are also not converted.
  • A few other spells are converted a bit suboptimally, and don't work quite the way they did in IWD. The current list is Power Word: Stun, Power Word: Kill, Blood Rage, Animal Rage, Chain Lightning, Static Charge, and Whirlwind.<.li>
  • The IWD dragon animation cannot straightforwardly be converted. We make do with the BG2 silver dragon animation.
  • The spell animations often have a sort of glow around them that is a bit unattractive.
  • Occasionally, selecting the map screen causes a crash. If you experience this, please let me know where (and if it's reproduceable) as I'm struggling to pin this down.
