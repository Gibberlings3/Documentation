Introduction

Rencontrez Tyris Flare, une amazone aux cheveux roux qui peut botter des fesses et lancer du feu !

Cela vous rappelle quelque chose ? C'est parce que, dans un élan de nostalgie, j'ai décidé de recréer un personnage du vieux jeu vidéo Golden Axe, sorti sur la Sega Megadrive (ou Genesis en Amérique) et les bornes d'arcade dans les années 90. Vous l'avez peut-être aussi vue par la suite dans Golden Axe : Beast Rider, mais mon interprétation du personnage est celle des anciens jeux.

Comment s'est-elle retrouvée dans Baldur's Gate II ? Pour le savoir, il va falloir jouer le mod !

Une fois dans votre groupe, Tyris aura des dialogues avec tous les PNJ de BioWare et pourra être romancée par n'importe quel personnage masculin humain/elfe/demi-elfe. Profitez-en !

Installation

Note sur les enhanced editions (EE)
Les enhanced editions continuent à être activement développées. Veuillez noter que chaque patch de mise à jour effacera votre configuration de mods actuelle ! Si vous êtes en pleine partie d'un jeu moddé, il est souhaitable de retarder l'installation du patch (si possible) car même après avoir réinstallé les mods, vous ne serez peut-être pas en mesure de continuer avec vos anciennes sauvegardes. Vous pouvez également copier l'ensemble du dossier du jeu dans un nouveau dossier qui pourra être moddé et qui ne sera pas modifié par les correctifs du jeu. Pour BG:EE, il est important que vous installiez le mod dans la langue dans laquelle vous jouez. Sinon, les dialogues du mod ne s'afficheront pas mais donneront des messages d'erreur.

Windows
Tyris Flare pour Windows est distribué sous forme d'archive auto-extractible et comprend un programme d'installation WeiDU. Pour l'installer, il suffit de double-cliquer sur l'archive et de suivre les instructions à l'écran.

Vous pouvez également extraire les fichiers dans votre répertoire de jeu à l'aide de 7zip ou WinRAR. Une fois correctement extraits, votre répertoire de jeu contiendra setup-tyrisflare.exe et le dossier tyrisflare. Pour l'installer, double-cliquez sur setup-tyrisflare.exe et suivez les instructions à l'écran.

Vous pouvez exécuter setup-tyrisflare.exe dans le dossier de votre jeu pour réinstaller, désinstaller ou modifier les composants du mod.

Mac OS X
Tyris Flare pour Mac OS X est distribué sous la forme d'une archive compressée et comprend un programme d'installation WeiDU.

Tout d'abord, extrayez les fichiers de l'archive dans votre répertoire de jeu. Une fois correctement extraits, votre répertoire de jeu contiendra setup-tyrisflare, setup-tyrisflare.command, et le dossier tyrisflare. Pour installer, double-cliquez sur setup-tyrisflare.command et suivez les instructions à l'écran.

Vous pouvez exécuter setup-tyrisflare.command dans votre dossier de jeu pour réinstaller, désinstaller ou modifier les composants du mod.

Linux
Tyris Flare pour Linux est distribué sous la forme d'une archive compressée et ne comprend pas d'installateur WeiDU.

Extrayez le contenu du mod dans le dossier du jeu que vous souhaitez modifier.

Téléchargez la dernière version de WeiDU pour Linux depuis WeiDU.org et copiez WeiDU et WeInstall dans /usr/bin. Ensuite, ouvrez un terminal, accédez au répertoire d'installation de votre jeu, exécutez tolower et répondez Y aux deux questions. Vous pouvez éviter d'exécuter la deuxième option (linux.ini) si vous l'avez déjà exécutée une fois dans le même répertoire. Pour gagner du temps, l'archive est déjà en lettres minuscules, il n'est donc pas nécessaire d'exécuter la première option (passer en lettres minuscules les noms de fichiers) si vous n'avez extrait que ce mod depuis la dernière fois que vous avez passé en lettres minuscules les noms de fichiers. Si vous n'êtes pas sûr, lancer tolower et choisir les deux options est la meilleure solution.

Pour installer, exécutez WeInstall tyrisflare dans votre dossier de jeu. Puis lancez wine BGMain.exe et commencez à jouer.

Remarque concernant la désinstallation complète
En plus des méthodes ci-dessus pour supprimer les composants individuels, vous pouvez désinstaller complètement le mod en utilisant setup-tyrisflare --uninstall à la ligne de commande pour supprimer tous les composants sans passer par les invites.

Contenu

Le mod contient plusieurs composants qui peuvent être installés indépendamment les uns des autres.

PNJ Tyris Flare

Il s'agit du composant principal qui ajoutera Tyris Flare à votre jeu Shadows of Amn ou Throne of Bhaal. On peut la trouver dans les collines d'Umar, pas loin de la caverne.

Portraits alternatifs
Nécessite le composant PNJ Tyris Flare

Ce composant à choix multiple vous permet d'installer en option l'un des sept portraits alternatifs pour Tyris.
