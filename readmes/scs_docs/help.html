Sword Coast Stratagems Component: Better Calls For Help

Roughly speaking, the help script works like this. Suppose some hostile creature - say, an Orc - runs into Player 3. He then Shouts something that could be translated as "I'm an Orc and I can see player 3!", and sets an internal variable that means "I'm running towards player 3!".

Any orc who hears that shout also sets that same internal variable. If orcs are counted as "intelligent" (like kobolds and gnolls, for instance, but unlike wolves and spiders), the new orc gives a different shout, which could be translated as "I'm an orc and I've just heard another orc shout for player 3!".

Any orc who hears the new shout sets the internal variable to say "I'm running towards player 3", but doesn't in turn shout - in playtesting it turned out that too many repeated Shouts just cause everything in the map to run towards the player, which leads to one exciting fight followed by a very long tedious exploration of an empty dungeon. There are a couple of exceptions though: the Bandit camp, for instance.

Finally, everyone who has an internal variable set to "run towards player 3" does so - if they can't see any hostiles.

There are fairly obvious ways to abuse this, but they don't tend to come up unless you actively try to set them up.

The full list of monsters affected, grouped into those groups which respond to each others' cries for help, is:

Stupid creatures:

  • Basilisks
  • Dogs
  • Spiders and ettercaps (arguably these creatures aren't "stupid", just not organised beyond small groups)
  • Wolves (also hear werewolf cries)
  • Wyverns

Intelligent creatures:

  • Bandits
  • The demon cultists in Ulgoth's Beard
  • Gibberlings
  • Gnolls
  • Hobgoblins
  • Kobolds
  • Ogres
  • Tasloi
  • Undead
  • Werewolves (also hear wolf cries)
  • Xvarts (and Ursa!)
  • & the mixed bag of (mostly human) creatures who use the Shout script

There is also a specific script for Slythe and Krystin, to make sure that they seek out the player once the fight starts (otherwise they have a tendency to get lost.)
