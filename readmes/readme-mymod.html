Installation

This is pretty boilerplate. Modify the text below as needed.

Windows
MyMod is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a MyMod folder and Setup-MyMod.exe in your BG2 folder. To install, simply double-click Setup-MyMod.exe and follow the instructions on screen.

Please run Setup-MyMod.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. MyMod is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder. If properly extracted, you should have a MyMod folder, Setup-MyMod, and Setup-MyMod.command in your BG2 folder. To install, simply double-click Setup-MyMod.command and follow the instructions on screen.
