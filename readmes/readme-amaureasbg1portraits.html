Overview

This mod offers alternative portraits for each of the Baldur's Gate NPCs. These portraits can also be used by the PC.

Compatibility

This mod is designed to work with Baldur's Gate with or without the Tales of the Sword Coast expansion. It may also be used with the Baldur's Gate conversion, Tutu, with or without the Tales of the Sword Coast expansion pack, or with BGT-WeiDU. It may be used on a Baldur's Gate II, Shadows of Amn, with or without the Throne of Bhaal expansion, however the only features that will have an effect in game are "Make all portraits available for PC" and those portraits for NPCs who also appear in BG2. This mod should be installed in your game directory, depending on which platform is used. The mod will automatically detect the platform.

There are no known incompatibilities with any mod, however any portrait modification mod will overwrite the portrait assignments of portrait mods installed previously.

Installation

Windows
Amaurea's BG Portraits is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have an Amaurea's BG Portraits folder and Setup-AmaureasBG1Portraits.exe in your game directory. To install, simply double-click Setup-AmaureasBG1Portraits.exe and follow the instructions on screen.

Please run Setup-AmaureasBG1Portraits.exe in your game directory to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. Amaurea's BG Portraits is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your game directory. If properly extracted, you should have an AmaureasBG1Portraits folder, Setup-amaureasbg1portraits, and Setup-amaureasbg1portraits.command in your BG2 folder. To install, simply double-click Setup-amaureasbg1portraits.command and follow the instructions on screen.

Linux
Extract the contents of the mod to the folder of the game you wish to modify. Download the latest version of WeiDU for Linux from WeiDU.org and copy WeiDU and WeInstall to /usr/bin. Following that, open a terminal, cd to your BG2 installation directory, run 'tolower' and answer Y to both queries.

You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you are unsure, running tolower and choosing both options is the safe bet.

Run WeInstall alternatives in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Contents

Please see this page for a complete listing of all the available portrait choices.

PC Portraits: This component allows the PC to use any of the portraits included in this mod. The naming convention is AMXXX#Y, where XXX is the first three letters of the NPC's name, and Y is the size of the portrait.

Preserve BioWare Portraits for use by the PC: this component copies the existing BioWare portraits to the portraits folder to allow the player to use them unchanged. To avoid overwriting, the portraits are renamed. For BG portraits, the naming convention is AMXXXBY, where XXX is the first three letters of the NPC's name, B is for BioWare, and Y is the size. For BG2 portraits, the naming convention is AMXXXNY, where XXX is the first three letters of the NPC's name, N is for "new," and Y is the size.

BG Portraits: This component offers up to five alternate portraits for each of the BG NPCs. Please note that if the player chooses the "Install All" option, the last portrait alternative will be installed for each NPC.
