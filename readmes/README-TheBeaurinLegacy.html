Overview

The Beaurin Legacy introduces an elven Enchanter/Thief and a hidden NPC to BGII:SoA, ToB, and BGII:EE. Minyae Beaurin can be found outside of the Crooked Crane in the City Gates. The mod includes custom portraits, friendship tracks, numerous banters and interjections, new items, encounters and quest(s), voicing for both NPCs (However, Minyae is fully voiced), a romance with a Bioware NPC, player-initiated dialogues, and custom friendship track themes.

Compatibility

This mod is designed to work with Baldur's Gate II: Enhanced Edition and the original Baldur's Gate II: Shadows of Amn and Throne of Bhaal by Bioware.

The mod should be compatible with other WeiDU mods.

Installation

Windows
The Beaurin Legacy is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a TheBeaurinLegacy folder and Setup-TheBeaurinLegacy.exe in your Tutu folder. To install, simply double-click Setup-TheBeaurinLegacy.exe and follow the instructions on screen.

Please run Setup-TheBeaurinLegacy.exe in your Tutu folder to reinstall, uninstall or otherwise change components.

Mac OS X
If the mod was previously installed, uninstall it before extracting the new version. The Beaurin Legacy is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your Tutu folder. If properly extracted, you should have a TheBeaurinLegacy folder, Setup-TheBeaurinLegacy, and Setup-TheBeaurinLegacy.command in your Tutu folder. To install, simply double-click Setup-TheBeaurinLegacy.command and follow the instructions on screen.

Linux
Extract the contents of the mod to the folder of the game you wish to modify. Download the latest version of WeiDU for Linux from WeiDU.org and copy WeiDU and WeInstall to /usr/bin. Following that, open a terminal, cd to your BG2 installation directory, run 'tolower' and answer Y to both queries. You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name. If you're unsure, running tolower and choosing both options is the safe bet.
Run WeInstall TheBeaurinLegacy in your game folder to install the mod. Then run wine BGMain.exe and start playing.

Contents

This modification adds Minyae, a Neutral Evil Elf Enchanter/Mage, to the game. Her statistics are as follows:

  • STR: 12
  • DEX: 19
  • CON: 11
  • INT: 17
  • WIS: 12
  • CHR: 16

Unlike most thieves and mages, she specializes in daggers and two weapon style. She also has a proficiency in darts.

Minyae includes a "friendship track" that continues through Throne of Bhaal, banters with all Bioware NPCs (including the Beamdog's four NPCs and a romance path for a certain Red Wizard), numerous interjections and extensive player-initiated dialogue.

In Throne of Bhaal, the hidden NPC also includes a "friendship track", banters with all Bioware NPCs, interjections, and extensive player-initiated dialogue.

Player-initiated Dialogue

The player can talk to Minyae or the Hidden NPC directly at any time in the game, no matter if the PC is on the friendship path or not. However, there is one specific friendship talk that must be initiated through the player-initiated dialogues to continue her friendship path in SoA. This dialogue will not be immediately available, you must wait approximately 45 minutes in real-time to initiate the talk.
